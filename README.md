Проект выполнен в рамках лабораторной работы по изучению взаимодействия клиентских приложений с серверным API. 
Основная цель - автоматизация работы с данными с использованием HTTP-запросов и библиотеки requests.
В качестве сессии пользователя использовались сессионные токены из Cookie, полученные путем перехвата запросов с помощью инструмента Burp Suite.
